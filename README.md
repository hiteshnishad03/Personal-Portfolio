# Personal Portfolio — Hitesh Nishad

A responsive personal portfolio website for **Hitesh Nishad**, a UI/UX & Website Designer. Built with plain HTML, CSS, and JavaScript — no frameworks, no build step.

🔗 **Live Demo:** _add your GitHub Pages / hosting link here_

## ✨ Features

- Fully responsive layout (desktop rail navigation + mobile dropdown nav)
- Smooth section navigation via a fixed side rail with active-section highlighting
- Sections: Home, About, Services, Projects, Skills, Testimonials, Contact
- Animated skill bars
- Clean, modern typography using Google Fonts (Poppins, Inter, JetBrains Mono)

## 🛠️ Built With

- **HTML5** — structure
- **CSS3** — styling & responsive layout
- **JavaScript** — navigation/interactivity

## 📁 Project Structure

```
Personal-Portfolio/
├── index.html      # Main page markup
├── style.css       # All styling
├── script.js       # Navigation & interactivity
└── README.md
```

## 🚀 Getting Started

No build tools or dependencies required.

1. Clone the repository
   ```bash
   git clone https://github.com/hiteshnishad03/Personal-Portfolio.git
   ```
2. Open the project folder
   ```bash
   cd Personal-Portfolio
   ```
3. Open `index.html` in your browser (or use the VS Code "Live Server" extension for auto-reload).
